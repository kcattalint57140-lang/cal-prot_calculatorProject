# Calorie & Protein Tracker

แอปบันทึกแคลและโปรตีนประจำวัน ตัวนี้ผมทำขึ้นมาเพราะอยากได้ตัวช่วยแทร็กอาหารที่เปิดบนมือถือได้ง่าย ๆ ไม่ต้องโหลดแอป แอปตัวนี้ผมมีพี่ Gem ของ (Gemini) ช่วยทำ

## แอปนี้ทำอะไรได้บ้าง?
* **จดแคล/โปรตีนรายวัน:** มี Progress Bar วิ่งตามเป้าหมาย ดูง่าย
* **ใส่สูตรเลขได้เลย:** ในช่องกรอกแคล พิมพ์ `100+50+120` ได้เลย เดี๋ยวแอปคำนวณรวมให้เอง 
* **มีปุ่มกดสลับโหมด:** Bulking (+400 แคล) หรือ Cutting (-500 แคล) ให้เป้าหมายปรับอัตโนมัติ
* **คำนวณ TDEE ในตัว:** มี Pop-up ให้กรอก นํ้าหนัก ส่วนสูง อายุ แล้วคำนวณหาค่าเผาผลาญ
* **เมนูโปรด:** อาหารไหนกินบ่อย เมนูไหนกินทุกวัน กดเซฟเป็น Preset ไว้ ครั้งหน้ามากดคลิกเดียวเพิ่มเข้าบันทึกวันนั้นได้เลย
* **กราฟสรุปนํ้าหนักและแคล:** ลิงก์ข้อมูลไปวาดกราฟเส้นกับกราฟแท่ง (ใช้ Chart.js) ดูแนวโน้มนํ้าหนักตัวเองได้
* **Dark Mode:** มีปุ่มสลับธีมมืด/สว่าง
* **Export / Import JSON:** ข้อมูลทั้งหมดเซฟลงเครื่องเรา (Local Storage) และสามารถกด Export เป็นไฟล์ `.json` เก็บไว้สำรองข้อมูลได้ 

## Tech Stack
* **HTML5 / CSS3:** เขียนโครงสร้างเอง และใช้พี่ gem ช่วยปรับแต่ง ตกแต่งให้ใช้งานง่าย และทำเป็น Responsive สเกลพอดีหน้าจอมือถือ
* **Vanilla JavaScript (ES6):** ลอจิกทั้งหมด ทั้งการคำนวณแคล ดึงค่าวันเวลา สลับโหมด จัดการข้อมูลหลังบ้าน
* **Chart.js:** ดึงมาใช้ทำกราฟสรุปผล
* **Local Storage:** ใช้เก็บข้อมูลผู้ใช้ไว้บนเบราว์เซอร์ ดึงข้อมูลเก่าขึ้นมาโชว์ได้โดยไม่ต้องเปิด Database
